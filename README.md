## 🚀 About the Project

**Quan AI** is a flagship web-based application designed to deliver an unparalleled conversational AI experience. Built with a heavy focus on zero-latency visual feedback, responsive mobile-first design, and native-feeling interactions, Quan AI bridges the gap between web apps and native applications. 

Developed and maintained by **Goorac Corporation**, this platform ensures high-fidelity markdown rendering, voice-to-text integration, and real-time cloud synchronization.

---

## ✨ Key Features

* **Real-Time Synchronization:** Chat histories and messages are synced instantly across all devices using Firebase Firestore.
* **Progressive Web App (PWA):** Fully installable on mobile and desktop devices with offline caching via Service Workers.
* **Advanced Markdown & Code Formatting:** Beautiful Mac-style code blocks, syntax highlighting, and fluid text rendering.
* **Voice Integration:** Native browser speech-to-text integration with an interactive voice visualizer.
* **Rich Media Support:** Drag-and-drop or upload image attachments with built-in client-side canvas compression.
* **Fluid UX/UI:** 60fps animations, haptic feedback integration, and smart auto-scrolling that eliminates layout jitter.
* **Secure Authentication:** Protected user sessions governed by Firebase Auth.

---

## 🛠 Tech Stack

* **Frontend:** HTML5, CSS3 (Custom variables, Flexbox/Grid, fluid animations), Vanilla JavaScript (ES6+).
* **Backend & Database:** Firebase Authentication, Cloud Firestore (v12.10.0).
* **Performance:** Base64 image caching, DOM paint optimizations, custom Service Workers.
